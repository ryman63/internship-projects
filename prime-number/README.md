# Простые числа
Определить является ли введенное число простым. Простое число – это натуральное число, имеющее ровно 
два различных натуральных делителя.

## Формат входных данных
int n; где 1 < n < 10000;

## Формат выходных данных
boolean: isPrime;

## Примеры:
### Входные данные
8

### Результат работы
false

### Входные данные
3

### Результат работы
true
