# Лесенка
С консоли вводится целое число - высота лесенки 
в строках, с помощью символа "_" необходимо вывести в консоль лесенку заданной высоты


## Формат входных данных
int n, где 1 < n < 20

## Формат выходных данных
строки

## Примеры:
### Входные данные
5

### Результат работы
```text
    _
   __
  ___
 ____
_____
