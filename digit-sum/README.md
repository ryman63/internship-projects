# Сумма цифр трехзначного числа
В консоль вводится трехзначное положительное целое число. 
Нужно вывести сумму его цифр.

## Формат входных данных
трехзначное положительное целое число
## Формат выходных данных
целое число

## Примеры:
### Входные данные
123    
### Результат работы
6
### Входные данные
555    
### Результат работы
15
